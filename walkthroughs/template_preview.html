Transform Per-Channel Normalisation ▶

Each channel is normalised independently using statistics computed over the training split. The operation clips first to the 1st–99th percentile range to suppress hot pixels, then applies a linear rescaling to $[0, 1]$.

$$\hat{r}_c = \frac{\text{clip}(r_c,\, p_1^c,\, p_{99}^c) - p_1^c}{p_{99}^c - p_1^c}$$
61lo, hi = np.percentile(rad[c][valid], [1, 99])
62rad[c] = np.clip(rad[c], lo, hi)
63rad[c] = (rad[c] - lo) / (hi - lo + 1e-8)

Result Evaluation against Hand Labels ▶

The assembled binary detection map is compared pixel-wise to hand-labelled scenes from the validation set. Precision, recall, and F1 are computed over valid (non-bowtie) pixels only, then aggregated across granules via micro-averaging.

Alternative path: Threshold Sweep

Rather than a fixed threshold, sweep_thresholds() evaluates every value in np.linspace(0.1, 0.9, 17) and returns the full precision–recall curve. The optimal operating point is chosen by maximising F1 on the validation split independently of the test set.
