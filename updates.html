Reflection June 5, 2026

Week 1 — Workflow

  • Started working with pi, and selected Claude Sonnet.
  • Described the AI research pilot and that I need a website; asked lots of questions on what I should provide and gave feedback that I want a folder structure instead of a single growing file with updates; lots of clarifying questions before providing details and information about me.
  • I did a lot of upfront manual writing (.md) to provide information since I wanted the agent to not have to guess information about me.
  • First website pass looked pretty good but I had lots of things I wanted to get tweaked; especially it looked like a website about me but it should be a website about my summer project.
  • One issue I ran into: it wasn't clear to me how to set up the local host web browser so it would stay there.
  • Cool: I used pi to compress images where I wasn't sure if it was able to — I wouldn't have known how to do it myself in a fast way.
  • It struggled to make certain text style adjustments, even when I gave examples of other text that was already in that style. Adjusting placement of some text and the banner took multiple iterations.
  • Making the plane easter egg and Instagram easter egg work also took multiple iterations — from it didn't work at all, to it broke part of the page, to it works but needs to be tweaked, to it finally looking like what I imagined.
  • This was a lot of upfront work — maybe because I pretty much knew how I wanted things — but I think adding things will be fast going forward once the structure is established.
